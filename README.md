# Berlin Mietspiegel Dashboard

Interactive tool for understanding Berlin rental prices, developed as a bachelor thesis project.

## Quick Start
```bash
pip install -r requirements.txt
python app.py
```

Visit `http://127.0.0.1:8050`

## What It Does

- Compare rental offers with official Mietspiegel benchmarks
- Explore rent levels across Berlin districts
- Look up location quality for specific addresses
- All in English for newcomers to Berlin

## Tech

Dash 3.3 | Plotly 6.5 | Pandas 3.0 | Dash Mantine Components

## Data

Based on Berlin Mietspiegel 2024 from [Berlin Open Data Portal](https://daten.berlin.de/)

## Note

Academic prototype - for official assessments, consult complete Mietspiegel documentation.
